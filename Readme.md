# 💼 Portfólio Pessoal

Este é um projeto de **portfólio pessoal** desenvolvido com HTML e CSS, com o objetivo de apresentar minhas habilidades técnicas, experiências profissionais, projetos desenvolvidos e certificações.

O site foi pensado como uma forma simples, organizada e visual de demonstrar minha evolução como desenvolvedor.

---

## 🚀 Funcionalidades

* 👨‍💻 Apresentação pessoal
* 🛠️ Listagem de linguagens e frameworks dominados
* 🏢 Experiência profissional detalhada
* 📂 Exibição de projetos desenvolvidos
* 📜 Seção de certificações
* 📞 Links de contato (redes sociais e e-mail)
* 📱 Layout responsivo básico

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** — Estrutura do site
* **CSS3** — Estilização e responsividade

## 💻 Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/portfolio.git
```

2. Acesse a pasta do projeto:

```bash
cd portfolio
```

3. Abra o arquivo `index.html` no navegador.

## 👨‍💼 Sobre Mim

Neste portfólio, apresento um resumo da minha trajetória como desenvolvedor, incluindo minha experiência como estagiário na empresa **2RP**, atuando no squad da **Case**.
